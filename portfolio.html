EnergieStudent

EnergieStudent connects entrepreneurial students with companies within the Energytransition in the Netherlands. This is a project I did with the goal to make an effective website for bringing in clients. The proces I used was as follows: first I made a design where I used customer journey and psychological 'influence' theories as a basis. Then I build the website and asked for a lot of feedback from clients and professionals. Lastly I implemented this feedback and put it online. Now on average 2 clients a week join EnergieStudent via the website, which is more than the goal we set at the beginning.
